# Tkinter To-Do App with JSON Persistence

A professional Python-based To-Do application featuring a Navy Blue theme, automated testing, and CI/CD integration.

## Features
- **CRUD Operations**: Add, complete, and delete tasks.
- **State Management**: Tasks are saved to `tasks.json` so they persist after closing the app.
- **Robustness**: Handles corrupted JSON files gracefully.
- **Automation**: Integrated with GitHub Actions for automated testing on every push.

## Tech Stack
- **Language**: Python 3.11+
- **GUI**: Tkinter
- **Testing**: Pytest
- **DevOps**: GitHub Actions (CI/CD)

## How to Run
1. Clone the repo.
2. Run `pip install -r requirements.txt`.
